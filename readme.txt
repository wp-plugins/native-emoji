Native Emoji Plugin, made the WordPress way.

== Description ==
This is not just a plugin, this is the plugin for use <strong>emoji</strong> in a native way. When activated you will see a new button in your wordpress editor, from there you will be able to include more than 1,000 emojis in to your post, pages, etc. Native Emoji is translation ready for all spanish versions and supports any custom post type.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Native Emoji'
3. Activate Native Emoji from your Plugins page.
4. Visit Any post type from your admin and click on the smiley button in the editor

= From WordPress.org =

1. Download Native Emoji.
2. Upload the 'Native Emoji' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Native Emoji from your Plugins page. 
4. Visit Any post type from your admin and click on the smiley button in the editor
